# nexdevstudioro

## Descriere

Proiectul **nexdevstudioro** reprezintă un punct de plecare pentru aplicații web moderne.

## Instalare

1. Clonează repository-ul:
   ```bash
   git clone https://github.com/nexdevstudioro/nexdevstudioro.git
2. Intră în directorul proiectului:
   ```bash
   cd nexdevstudioro
3. Instalează dependențele:
   ```bash
   npm install

## Utilizare

- Pentru a porni serverul de dezvoltare:
  ```bash
  npm start
- Pentru a construi aplicația pentru producție:
  ```bash
  npm run build
- Pentru a rula testele:
  ```bash
  npm test
